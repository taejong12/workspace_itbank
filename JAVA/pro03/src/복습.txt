1. 클래스, 함수, 변수 이름 만들기
	1) 클래스 이름은 반드시 첫글자는 대문자로 쓴다.
	2) 함수 또는 변수 이름은 소문자로 시작한다.
		- 클래스 함수 변수의 단어를 연결 해서 만들 경우 
			다음 단어의 첫 글자는 대문자로 한다.
		- class : Title, TitleScore, SpeedScore, CarSpeed
		- 함수, 변수 : title, titleScore, speedScore, carSpeed
	3) 이름은 누구나 보면 이해되는 이름을 만든다.
	4) 한글로 변수를 만들 수 있으나 사용하지 않는다.
	5) 숫자와 기호로 시작 하면 안된다.($, _ 는 제외)
	
2. 변수의 데이터 타입
	1) 정수형 : int : byte, short, char, long
	2) 실수형 : double : float
	3) 문자열 : String
	- int, double, String 만 기억할것
	- int, double 은 전부 소문자, String 은 시작이 대문자
	
3. 변수 선언
	1) [데이터타입] [변수명] = [값 또는 연산]
	2) 데이터타입 : int, double, String 중 하나 선택
	3) 변수명은 다른 사람도 이해할 수 있는 이름을 만들기
	4) 값 또는 연산에서 사용되는 데이터 타입으로 지정한다.
	
4. 소스 코드 작성
public class [클래스명] {
	public static void main(String[] args) { #
		1. 소스코드는 반드시 main 중괄호 안에 써야 한다.
			- 이 표에서 # ~ # 사이에 작성
		2. 문장이 끝나면 반드시 ; (쎄미콜론) 을 써야 한다.
		3. 실행은 Ctrl + F11 으로 실행 한다.
		4. 소스코드 파일에 빨간 밑줄이 있으면 실행되지 않는다.
			- 만일 실행이 되면 이전에 실행 되었던 class 다.
		5. 빨간 밑줄에 마우스를 가져다 대면 처리 방법이 나옴
		6. 노란 밑줄은 경고이기 때문에 실행에는 문제가 없다.
		7. Ctrl + Spacebar 를 이용하여 자동완성을 많이 사용
			- 사용할 패키지를 자동으로 호출해 준다.
			- 오타를 줄일 수 있다.
			- 어떤 메서드가 있는 지 확인 가능하다.
			- 메서드 선택에 도움을 준다.
	# }
}	
	
5. 연산자
	1) 산술연산자 : + , - , * , /(몫) , %(나머지)
	2) 비교연산자 : ==, !=, >, <, >=, <=
	3) 대입연산자 : =
	4) 논리연산자 : &&(and), ||(or)
	5) 증감연산자 : ++, --
		
6. 출력
	1) System.out.println("출력 내용")
		- 출력 내용을 출력한 후 한칸 내림
	2) System.out.print("출력 내용")
		- 출력 내용을 출력한 후 아무일도 없음
	3) System.out.printf("출력 내용")
		- 출력 내용을 출력한 후 아무일도 없음
		- format : 형식을 가지는 출력문이다.
		- %s : 문자열
		- %d : 정수
		- %f : 실수 - %.2f : 소수점의 자릿수 지정
		- %c : 단일 문자
		- 출력할 형식과 출력할 데이터의 수는 같아야 한다.
		- ,(콤마) 를 이용해서 데이터를 추가한다.
		
7. 입력
	1) Scanner 를 이용한 방법을 제일 많이 사용함.
	2) Scanner.next() : 문자열 - String
	3) Scanner.nextInt() : 정수형 - int
	4) Scanner.nextDouble() : 실수형 - double
	5) 데이터를 입력 받기 위해서는 변수가 필요하다.
		- 변수의 데이터 타입은 입력받는 데이터와 동일해야 한다.
